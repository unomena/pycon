PyconZA October 2012 Schedule

We are excited to announce the draft programme for the very first PyCon in Africa, PyConZA 2012. The theme is getting stuff done with Python -- in Africa! Throughout Africa people are doing things with Python -- building massive radio telescopes, creating mobile health systems, running businesses, making websites, writing games, connecting people, doing research, designing interpreters, crunching data, earning money and just coding for fun!

If you’re a newcomer looking for tutorials to get you started, or wanting to be inspired by what others around you are achieving with Python, our programme has both for you. If you’re a Python guru, come to share insights, ideas and coffee -- there will be plenty of other experienced folk around -- or, if you’re particularly excited about something, offer us a talk!

The program below is a first draft, and will be continuously updated. The final schedule will be announced closer to the event.

Core Python:
Getting started with Python, unit testing, Python performance, Python 3

Web development:
Django, GeoDjango, Flask, Plone

Science, engineering and data crunching:
Scipy, Numpy, Pypy, Pandas, spatial data infrastructures

Mobile:
Android applications in Python, text messaging, Django for mobi sites, embedded Python

Backend Python:
Twisted, managing cloud servers (Fabric, Boto), monitoring (Diamond, Carbon, Graphite), writing scalable Python programs

Data storage:
SQLAlchemy, SQLObject, NoSQL datastores

Community development:
Getting involved with Python projects, PyWeek

People doing things with Python:
Radio telescopes (SKA), mobile Health projects (UNICEF, Praekelt)

If you'd like to speak on any of these topics or have any topics you'd like us to find presenters for, email team@za.pycon.org. If you’re a Python guru, come share insights, ideas and coffee -- there will be plenty of other experienced folk around -- or, if you’re particularly excited about something, offer us a talk!
